### 2. User Registration

#### URL

- `POST` /api/v1/user/_register

Description:

ContentType：`application/json`

#### Request Parameters

##### Body Parameter

| Name | Type | Required | Max Length | Description | Example |
| --- | --- | --- | --- | --- | --- |
| passwd | string | Y | - |  | 2 |
| phone | string | N | - |  | 1111 |
| nickname | string | N | - |  | 2 |
| email | string | N | - |  | 2222 |
| username | string | Y | - |  | 5555 |

#### Request Example

```
{
    "passwd": "2",
    "phone": "1111",
    "nickname": "2",
    "email": "2222",
    "username": "5555"
}
```

#### Response Parameters

None

#### Response example

```
{
    "msg": "User registration successful!",
    "code": 0,
    "data": ""
}
---------------------------
{
    "code": 420,
    "msg": "The user already exists!"
}
```

#### Error Codes

None
